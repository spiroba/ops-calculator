Описание и методика определения размера личного бонуса     6.1.	Личный бонус устанавливается для руководителя группы ОПС, работников неспециализированных ОПС (здесь и далее по тексту Параметров включая ЦВПП), ППС, МГ.     6.2.	Для начальника отделения/центра в ОПС/ППС с операторами, руководителя группы ОПС размер личного бонуса (ЛБ) определяется по формуле:     ЛБ = ?КЛБ_1 + ?КЛБработники? 3 % ? t / T,     где     ?КЛБ_1 – сумма компонентов личного бонуса начальника отделения/центра, руководителя группы ОПС за операции типа 14, установленные для соответствующей должности, руб.;     ?КЛБработники – сумма компонентов личного бонуса иных работников ОПС/ППС в зоне ответственности (кроме работников в должности начальника отделения/центра, руководителя группы ОПС) за операции всех типов, за исключением операций, за которые компонент личного бонуса предусмотрен для самого начальника отделения/центра, руководителя группы ОПС;     t5 – время, отработанное начальником отделения/центра, руководителем группы ОПС в отчетном периоде, час.;     T6 – количество рабочих часов, установленных начальнику отделения/центра, руководителю группы ОПС, час.     6.3.	Для начальника отделения/центра в ОПС/ППС без операторов, а также должностей, не указанных в п. 6.2 Параметров, размер личного бонуса (ЛБ) определяется по формуле:     ЛБ = ?КЛБ_1 + (?КЛБ_2 + ?КЛБ_3 + ?КЛБ_4) ? Кэффективность,     где     ?КЛБ_1, ?КЛБ_2, ?КЛБ_3, ?КЛБ_4 – сумма компонентов личного бонуса работника за операции типов 1–4, установленные для соответствующей должности/подразделения (при наличии), руб.;Кэффективность – коэффициент «Эффективность ОПС», равный:     * 1,3 – в случаях, если выполнение плана по КПЭ «Выручка ОПС» равно 100 % или более;     * 1 – в иных случаях.6.4.	По виду учета количества/объема операции делятся на:     –	индивидуальные (ИНД), учет которых осуществляется до конкретного работника (идентификатором является УИД сотрудника, а в отдельных случаях, предусмотренных Параметрами, – номер телефона). Устанавливаются для работников неспециализированных ОПС, ППС, МГ;     –	командные (КОМ), учет которых осуществляется до конкретного ОПС/ППС (идентификатором является индекс подразделения). Устанавливаются для работников неспециализированных ОПС, ППС.     6.5.	По способу расчета размера компонента личного бонуса (КЛБ) операции делятся на типы, для каждого из них устанавливается формула определения компонента личного бонуса (КЛБ):     * тип 1: КЛБ_1 = N ? СВ,     * тип 2: КЛБ_2 = N ? СВ ? Ксервис,     * тип 3: КЛБ_3 = N ? СВ ? Кскорость_прием,     * тип 4: КЛБ_4 = N ? СВ ? Кскорость_вручение,,     где     N – количество/объем операций. Для операций индивидуального вида – операции, проведенные конкретным работником, для операций командного вида – общее количество операций, проведенных в конкретном ОПС/ППС с учетом положений п. 6.6 Параметров;     СВ – ставка вознаграждения;     Ксервис – командный коэффициент «Сервис», равный:     * 1,5 – в случаях, если фактическое значение по КПЭ «Уровень качества клиентского сервиса (CSI канала «Сеть») по ОПС (далее – факт CSI) равно 95 баллов или более;     * 1 – в случаях, если факт CSI менее 95 баллов, но более или равен 90 баллов, а также в случае, если факт CSI не определен;     * 0,5 – в случаях, если факт CSI менее 90 баллов;     Кскорость_прием – командный коэффициент «Скорость приема», равный:     * 1,5 – в случаях, если среднее время приема РПО (всех типов) в ОПС равно плановому (доведение плановых значений осуществляется одновременно с плановыми значениями по КПЭ в соответствии с п. 5.2 Положения о премировании) или менее планового;     * 0,5 – в иных случаях (включая отсутствие плана);     Кскорость_вручение – командный коэффициент «Скорость вручения», равный:     * 1,5 – в случаях, если среднее время вручения РПО (всех типов) в ОПС равно плановому (доведение плановых значений осуществляется одновременно с плановыми значениями по КПЭ в соответствии с п. 5.2 Положения о премировании) или менее планового;     * 0,5 – в иных случаях (включая отсутствие плана).     6.6.	Для всех операций командного вида N конкретного работника корректируется (умножается) на коэффициент отработанного времени (Kt), который рассчитывается по формуле:Kt = t / Сумм.t,     где     t – отработанное время конкретного работника, для должности которого предусмотрено вознаграждение за операцию, час.;     Сумм.t – сумма отработанного времени (t) всех работников конкретного ОПС/ППС, для должностей которых предусмотрено вознаграждение за операцию, час.     Трактовка t и T соответствует п. 8.3 Положения о премировании.     6.7. Распределение операций по типам и должностям работников, размер ставок вознаграждения установлены в разделе 7 Параметров. Если должность/подразделение не соответствуют критериям, указанным в разделе 7 Параметров, компонент личного бонуса работника равен 0 руб.     6.8. Доведение плановых значений по операциям и мониторинг промежуточных значений не проводится, если иное не предусмотрено п. 6.5 и разделом 7 Параметров.     6.9. Процедура подведения итогов по операциям осуществляется в соответствии с п. 7.1 Положения о премировании.     6.10. Фактические данные направляются ответственным работником аналитического центра Департамента аналитической поддержки адресатам по формам в соответствии с разделом 8 Параметров.     6.11.
